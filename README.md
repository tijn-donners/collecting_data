# Shopping List 
This dummy dataset contains a shopping list.

This data has been generated by Claude Sonnet 4.5 from [Anthropic](https://www.anthropic.com/)

<img src="shopping.jpg" alt="Shopping List" width="300">

## Attributes
|Attributes|Description|
|:---------|-----------:|
|Item      |Name of Item|
|Category  |Category of Item|
|Quantity  |Number of items needed|
|Unit Price|Price of the item|
|Store     |Store where item is sold|
